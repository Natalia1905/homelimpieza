
## Instalation 

1. Clone repository
```shell
git clone https://github.com/flatlogic/light-blue-react-template.git
```
2. Get in the project folder
```shell
cd light-blue-react-template
```
3. Install dependencies via npm or yarn
```shell
npm install
```
```shell
yarn
```
